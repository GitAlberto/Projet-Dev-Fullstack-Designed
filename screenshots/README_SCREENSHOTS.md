# Note sur les Captures d'Écran

En raison d'une limitation technique de l'environnement de développement, les captures d'écran n'ont pas pu être générées automatiquement.

## Instructions pour Capturer les Screenshots

Pour compléter la documentation, veuillez capturer les screenshots suivants de l'application en fonctionnement :

### À Capturer

1. **01_login.png** - Page de connexion
   - Accéder à `http://localhost:5173`
   - Capturer la page de login

2. **02_register.png** - Page d'inscription
   - Cliquer sur le lien "S'inscrire"
   - Capturer le formulaire d'inscription

3. **03_reviews_user.png** - Liste des avis (utilisateur normal)
   - Se connecter avec `user@example.com` / `password`
   - Capturer la liste des avis

4. **04_reviews_admin.png** - Liste des avis (admin)
   - Se connecter avec `admin@example.com` / `password`
   - Capturer la vue admin de tous les avis

5. **05_create_review.png** - Formulaire de création
   - Cliquer sur "Créer un avis"
   - Capturer le formulaire vide

6. **06_analysis_result.png** - Résultat d'analyse
   - Soumettre un avis (ex: "Excellent produit !")
   - Capturer la page de résultat avec l'analyse IA

7. **07_dashboard_admin.png** - Dashboard admin
   - En tant qu'admin, accéder au Dashboard
   - Capturer les statistiques et graphiques

8. **08_mobile_login.png** - Vue mobile login
   - Redimensionner le navigateur à 375px de large
   - Capturer la page de login responsive

9. **09_mobile_reviews.png** - Vue mobile liste avis
   - En mode mobile (375px)
   - Capturer la liste des avis

### Outil Recommandé

Utilisez l'outil de capture d'écran de votre choix :
- **Windows:** Outil Capture d'écran (Win + Shift + S)
- **Extension navigateur:** GoFullPage (pour captures entières)
- **Outil professionnel:** Lightshot, ShareX
